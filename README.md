## Description

Doctor AI collects patient data through a questionnaire and provides recommendations to patients and doctors based on their health.

## Features

- Gathers lifestyle and health data through APIs
- Analyzes trends in insulin sensitivity
- Sends clear, prioritized suggestions
- Works with clinician dashboards

## Technologies

Built with TypeScript, CSS, and HTML.

## Setup

Run `npm i` to install the dependencies.

Run `npm run dev` to start the development server.
